Twenty hands-on lessons that translate what you already know from Java into TypeScript. Each lesson shows a familiar Java idiom, gives you a small problem to solve in TypeScript, and checks your answer right in the browser — no toolchain required. Your progress is saved locally, and a reference solution is one click away if you get stuck.

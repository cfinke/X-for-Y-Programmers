Twenty hands-on lessons that translate what you already know from Rust into Ruby. Each lesson shows a familiar Rust idiom, gives you a small problem to solve in Ruby, and checks your answer by running it on Wandbox (internet connection required). Your progress is saved locally, and a reference solution is one click away if you get stuck.

    Rust ⇄ Ruby cheat sheet

    A comprehensive quick reference pulled together from all 20 lessons — every idiom mapped between the two languages, in one place.

Twenty hands-on lessons that translate what you already know from Python into PHP. Each lesson shows a familiar Python idiom, gives you a small problem to solve in PHP, and checks your answer by running it on Wandbox (internet connection required). Your progress is saved locally, and a reference solution is one click away if you get stuck.

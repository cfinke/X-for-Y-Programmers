Twenty hands-on lessons that translate what you already know from JavaScript into Rust. Each lesson shows a familiar JavaScript idiom, gives you a small problem to solve in Rust, and checks your answer by compiling and running it on the Rust Playground (internet connection required). Your progress is saved locally, and a reference solution is one click away if you get stuck.

    JavaScript ⇄ Rust cheat sheet

    A comprehensive quick reference pulled together from all 20 lessons — every idiom mapped between the two languages, in one place.

Twenty hands-on lessons that translate what you already know from JavaScript into C#. Each lesson shows a familiar JavaScript idiom, gives you a small problem to solve in C#, and checks your answer by compiling and running it on Wandbox (internet connection required). Your progress is saved locally, and a reference solution is one click away if you get stuck.

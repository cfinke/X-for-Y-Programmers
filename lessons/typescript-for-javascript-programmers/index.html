Twenty hands-on lessons for JavaScript programmers: you already know the runtime — these lessons teach the type layer TypeScript adds on top. Each lesson shows familiar JavaScript, gives you a small typing problem to solve, and checks your answer right in the browser — no toolchain required. Your progress is saved locally, and a reference solution is one click away if you get stuck.

    JavaScript ⇄ TypeScript cheat sheet

    A comprehensive quick reference pulled together from all 20 lessons — every idiom mapped between the two languages, in one place.

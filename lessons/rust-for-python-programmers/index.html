Twenty hands-on lessons that translate what you already know from Python into Rust. Each lesson shows a familiar Python idiom, gives you a small problem to solve in Rust, and checks your answer by compiling and running it on the Rust Playground (internet connection required). Your progress is saved locally, and a reference solution is one click away if you get stuck.

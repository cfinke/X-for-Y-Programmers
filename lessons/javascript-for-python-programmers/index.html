Twenty hands-on lessons that translate what you already know from Python into JavaScript. Each lesson shows a familiar Python idiom, gives you a small problem to solve in JavaScript, and checks your answer right in the browser — no toolchain required. Your progress is saved locally, and a reference solution is one click away if you get stuck.
